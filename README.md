# MORTAL FURY 🥊

2D файтинг в стиле Mortal Kombat — браузерная игра на Canvas.

## Запуск
```bash
cd ~/Downloads/mortalgame
python3 -m http.server 8080
# Открыть http://localhost:8080
```
Или просто открыть `index.html` в браузере.

## Управление (ПК)

| Клавиша | Действие |
|---------|----------|
| **A/D** или **←/→** | Движение |
| **W** / **Space** / **↑** | Прыжок |
| **S** / **↓** | Блок |
| **J** / **Z** | Удар кулаком |
| **K** / **X** | Удар ногой |
| **L** / **C** | Спецудар (нужно 50% шкалы) |
| **U** / **V** | Апперкот (подбрасывает) |

## Управление (Мобильный)
- **D-pad** — движение
- **👊** — удар
- **🦶** — пинок
- **🔥** — спецудар
- **🛡️** — блок

## Персонажи

| Боец | Стиль | Спецудар |
|------|-------|----------|
| **SHADOW** | Скорость + Огонь | Огненный шар |
| **FROST** | Лёд + Сила | Ледяной заряд (замораживает!) |
| **DRAGON** | Баланс | Огненный дракон |

## Механики
- ⚡ **Комбо** — 3+ удара подряд = комбо бонус
- 🛡️ **Блок** — снижает урон на 85%
- 🥶 **Заморозка** — Frost может заморозить врага
- ⬆️ **Апперкот** — подбрасывает в воздух
- 💥 **Slow-mo** — замедление при мощных ударах
- 📸 **Screen shake** — тряска экрана при попаданиях
- 🏆 **Раунды** — best of 3 (2 победы для выигрыша)
- ⏱️ **Таймер** — 60 секунд на раунд
